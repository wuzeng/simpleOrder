需求说明
========

目的
----

为老客户提供快捷下单订货渠道，方便对订单的管理，并提高客户满意度。

此版本特点
---------

* 为小用户群，低访问量。买家目前使用人数在100人以下，卖家一人一处登录。
* 功能简单，仅有下订单，无需线上支付。
* 此项目免费开源。

功能分析
-------

此项目包含两个部分：

### 前台

* 商品列表
* 订单详情
* 订单列表
* 个人信息

### 后台

* 商品管理
* 订单管理

此项目有两个角色：

* 买家  `拥有前台的浏览权限`
* 卖家 `拥有所有权限`


前台——商品列表
=============

展示实时数据，为客户提供购买参考
